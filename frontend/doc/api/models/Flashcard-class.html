Flashcard

Flashcard class

Modelová trieda pre jednotlivú flashcard.

Obsahuje:

  • ID kartičky,
  • ID setu, do ktorého patrí,
  • názov kartičky,
  • prednú a zadnú stranu (text),
  • typ dát (text, picture).

Táto trieda sa používa pri načítavaní a zobrazovaní kartičiek.
